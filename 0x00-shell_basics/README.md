## This is the readme file for the 0x00 shell basics
This file describes what each script in this directory contains.

File 0-current_working_directory - Command to print the current working directory

File 1-listit - Command to display the content list of the current working directory

File 2-bring_me_home - Command to change the working directory to the user's working directory

File 3-listfiles - Command to display current directory contents in long format

File 4-listmorefiles - Command to display to display current directory contents including hiden files (starting with .), using long format

File 5-listfilesdigitonly - Command to display current directory contents according the following rules 1) Long format. 2) User and group IDs displayed numerically. 3) Display hidden files (starting with .)

File 6-firstdirectory - Command to create a directory named my_first_directory in the /tmp/ folder

File 7-movethatfile - Command to move the file betty from /tmp/ to /tmp/my_first_directory

File 8-firstdelete - Contains command to delete the file betty in /tmp/my_first_directory

File 9-firstdirdeletion - Contains command to delete the directory my_first_directory in /tmp/

File 10-back - Contains command to change the working directory to the previous one

File 11-list - Contains comand to list all files in the current directory (including ones with names beginning with period) in the current directory, and the parent of the working directory, and /boot directory in long format, in this order

File 12-file_type - Contains command to print the type of the file name iamafile in the /tmp/ folder

File 13-symbolic_link - Contains command to create symbolic link to /bin/ls named __ls__

File 14-copy_html - Contains command to copy all the html files from the working directory to the parent of the working directory, buty only copy files that did not exist in the parent directory, or were newere than the version in the parent directory
